## 🚀 Projects

### 1. [Multimodal Retrieval-Augmented Generation (RAG) System](https://github.com/Mounika-Alwar/Multimodal-RAG-System)
- Built a cross-modal RAG pipeline that ingests **text, images, PDFs**, extracts content (PyMuPDF, BLIP), embeds using **CLIP/text-embeddings**, and stores vectors in **FAISS** for fast semantic search.  
- Designed a **FastAPI backend** with endpoints for upload, query, and FAISS reset, supporting TinyLlama and Gemini-powered generation.  
- Implemented metadata-rich chunking, cross-modal retrieval, and **Dockerized deployment workflows**.

### 2. [Snow Sense – LLM-Assisted Geospatial Snow Analysis](https://github.com/Mounika-Alwar/SnowSense)
- Developed a geospatial analytics workflow for **Sentinel-2 imagery**, computing NDSI and classifying snow.  
- AI Agent Mode using **Gemini** to auto-fetch regions, analyze imagery, and generate outputs from natural language prompts.  
- Automated retrieval, processing, and snow-coverage estimation for regions like **Siachen, Tibetan Plateau, Alps**.

### 3. [GPUlympics – GPU Performance Prediction Dashboard](https://github.com/Mounika-Alwar/GPUlympics)  
**Live App:** [GPUlympics Dashboard](https://mounika-alwar-gpulympics-frontendfrontend-j3d29g.streamlit.app/)  
- Interactive dashboard using **Streamlit + Plotly** to predict GPU efficiency and energy usage via **RandomForest MultiOutputRegressor**.  
- Real-time comparisons across GPUs with **REST API backend** and Dockerized deployment.

### 4. [Vibe Matcher – AI Fashion Recommendation Prototype](https://github.com/Mounika-Alwar/Vibe-Matcher)  
**Live App:** [Vibe Matcher](https://mounika-alwar-vibe-matcher-frontendapp-rrnpn9.streamlit.app/)  
- Semantic recommendation engine using **SentenceTransformer embeddings** and **FAISS** vector search.  
- Fast prototyping pipeline demonstrating embedding-based ranking, top-k retrieval, and similarity-driven recommendations.

### 5. [Autolytica – AI-Powered Business Analytics Platform](https://github.com/Mounika-Alwar/Autolytics)
